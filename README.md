# SCCT_Versus_Reloaded_Editor
This is an unofficial patch for the SCCT Versus Level Editor.

[![Watch the video](https://img.youtube.com/vi/xxnMBnKBSjY/maxresdefault.jpg)](https://youtu.be/xxnMBnKBSjY)
### [See changes on YouTube](https://youtu.be/xxnMBnKBSjY)

# Current Features
* Removes huge delay when clicking to select items
* Allows audio files above 200kb to be used
* Doubles lightmap resolution and enhances quality
* Fixes lighting (Windows only)

# Install
* Unzip the files into your game's `/System` folder where `ChaosTheory_Editor.exe` should be found
* Run `Reloaded_Editor.exe` and check the version in the title bar says 1.1
